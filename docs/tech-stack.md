# 技术栈说明

## 前端技术栈

| 技术 | 版本 | 选型理由 |
|------|------|----------|
| React | 18.3.1 | 生态成熟、社区活跃、组件化开发 |
| Vite | 5.1.0 | 快速开发、HMR 体验好、构建速度快 |
| Tailwind CSS | 3.4.1 | 原子化 CSS、快速开发、无需手写 CSS |
| Zustand | 4.5.0 | 轻量级状态管理、简单易用、无 Provider 包装 |
| React Router DOM | 6.22.0 | React 官方推荐路由方案 |
| TypeScript | 5.3.3 | 类型安全、IDE 支持好、减少运行时错误 |

## 后端技术栈

| 技术 | 版本 | 选型理由 |
|------|------|----------|
| Fastify | 4.26.0 | 高性能、Schema 验证、插件系统完善 |
| Prisma | 5.10.0 | TypeScript 原生、类型安全、迁移工具强大 |
| MySQL | 8.0 | 最流行的关系型数据库、云服务支持好 |
| JWT | - | 无状态认证、跨服务友好 |
| TypeScript | 5.3.3 | 前后端统一语言、类型安全 |

## 开发工具

| 工具 | 用途 |
|------|------|
| pnpm | 包管理器（必须使用） |
| Vitest | 单元测试 |
| tsx | TypeScript 直接运行 |
