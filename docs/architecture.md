## 2. 模块划分

### 2.1 前端模块

| 模块 | 功能 | 页面 |
|------|------|------|
| auth | 认证 | 登录页 |
| family | 家庭管理 | 创建家庭、加入家庭、成员管理 |
| record | 记账 | 记账页、账目列表、账目详情 |
| category | 分类管理 | 分类管理页 |
| stats | 统计 | 统计报表页 |

### 2.2 后端模块

| 模块 | 功能 | API |
|------|------|-----|
| auth | 认证 | 登录、验证码 |
| user | 用户管理 | 用户信息 CRUD |
| family | 家庭管理 | 家庭 CRUD、成员管理 |
| record | 记账 | 账目 CRUD |
| category | 分类管理 | 分类 CRUD |

## 3. 数据流

```
用户操作 → 组件 → Service → API → 后端 Controller → Service → Prisma → 数据库
```

## 4. 认证流程

```
1. 用户登录 → 后端验证 → 生成 JWT Token
2. 前端存储 Token（localStorage）
3. 后续请求携带 Token（Authorization: Bearer <token>）
4. 后端中间件验证 Token → 解析用户信息 → 继续处理请求
```
