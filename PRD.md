# 家庭记账本 - 产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品定位
一款简洁高效的家庭共享记账 H5 应用，让家庭成员轻松协作记账，实现家庭财务透明化管理。

### 1.2 产品愿景
成为家庭财务管理的首选工具，让每个家庭都能轻松掌握收支状况。

### 1.3 目标用户
- 25-45岁家庭主理人
- 关注家庭财务状况的用户
- 希望家庭成员共同参与记账的用户

### 1.4 使用场景
- 日常消费后即时记录
- 月底家庭对账
- 家庭财务规划讨论
- 查看收支趋势分析

## 二、用户分析

### 2.1 用户画像

| 维度 | 描述 |
|------|------|
| 年龄 | 25-45岁 |
| 身份 | 家庭主理人、夫妻双方 |
| 能力 | 有一定互联网使用能力，熟悉手机操作 |
| 特征 | 关注家庭财务，希望家庭成员共同参与记账 |

### 2.2 使用场景

| 场景 | 描述 | 频次 |
|------|------|------|
| 日常记账 | 购物、餐饮、交通等消费后即时记录 | 高频（每日多次） |
| 月底对账 | 家庭成员一起查看本月收支情况 | 中频（每月1次） |
| 财务规划 | 根据历史数据规划下月预算 | 中频（每月1次） |
| 分类查看 | 查看某类消费的详细记录 | 中频（每周数次） |

### 2.3 用户痛点

| 痛点 | 描述 | 严重程度 |
|------|------|----------|
| 广告干扰 | 现有记账App广告多，影响使用体验 | 高 |
| 功能臃肿 | 功能过多，学习成本高，核心记账功能被淹没 | 高 |
| 家庭共享难 | 家庭成员记账分散，难以汇总统计 | 高 |
| 隐私顾虑 | 数据存储在云端，担心隐私泄露 | 中 |
| 平台限制 | 需要下载App，不同手机系统体验不一致 | 中 |

### 2.4 用户期待

- 简单易用，3秒完成记账
- 家庭成员共享账本，数据实时同步
- 数据安全，隐私有保障
- 跨平台使用，无需安装App

## 三、竞品分析

### 3.1 竞品对比

| 竞品 | 类型 | 核心功能 | 优点 | 缺点/痛点 | 可借鉴 |
|------|------|----------|------|-----------|--------|
| 钱迹 | 直接竞品 | 记账、分类统计、预算管理、多账本 | 界面简洁、无广告、数据可导出 | 家庭共享功能较弱、需付费解锁高级功能 | 简洁UI设计、分类体系 |
| 随手记 | 直接竞品 | 记账、报表、预算、投资理财、家庭账本 | 功能全面、支持家庭共享、生态丰富 | 广告多、功能臃肿、学习成本高 | 家庭账本共享机制 |
| 挖财 | 直接竞品 | 记账、理财、社区 | 社区活跃、理财内容丰富 | 广告多、隐私顾虑 | 社区运营思路 |
| 叨叨记账 | 直接竞品 | 记账、角色互动 | 趣味性强、适合年轻人 | 功能较单一、不适合严肃记账 | 趣味化设计元素 |
| YNAB | 间接竞品 | 预算驱动记账 | 理念先进、预算控制强 | 价格昂贵、学习曲线陡 | 预算优先理念 |
| Excel/Google Sheets | 间接竞品 | 自由记账 | 完全自定义、免费 | 无移动端体验、操作繁琐 | 数据导出能力 |

### 3.2 差异化机会

1. **家庭共享体验优化** - 现有产品家庭共享功能入口深、体验差
2. **轻量化设计** - 避免功能臃肿，专注记账核心体验
3. **Web H5 跨平台** - 无需安装，微信/浏览器直接使用
4. **隐私优先** - 本地存储为主，可选云同步

### 4.2 核心功能说明

#### 4.2.1 用户注册/登录

**功能描述：** 用户通过手机号或微信授权注册/登录

**业务规则：**
- 支持手机号+验证码登录
- 支持微信一键授权登录
- 首次登录自动创建个人账户

**交互说明：**
- 登录页面简洁，突出第三方登录
- 验证码60秒倒计时

#### 4.2.2 家庭管理

**功能描述：** 创建家庭、邀请成员、管理成员权限

**业务规则：**
- 每个用户只能创建/加入一个家庭
- 创建者自动成为管理员
- 管理员可邀请/移除成员
- 通过邀请码或链接邀请成员

**交互说明：**
- 首页引导创建或加入家庭
- 邀请链接支持微信分享

#### 4.2.3 快速记账

**功能描述：** 快速记录收入/支出

**业务规则：**
- 必填项：金额、类型（收入/支出）、分类
- 选填项：备注、日期时间
- 默认使用当前时间
- 支持选择记账人（家庭模式下）

**交互说明：**
- 首页底部悬浮记账按钮
- 金额输入自动弹出数字键盘
- 分类以图标+文字展示，支持滑动选择
- 记账成功后显示成功提示

#### 4.2.4 分类管理

**功能描述：** 管理收入/支出分类

**业务规则：**
- 预设常用分类（餐饮、交通、购物等）
- 支持自定义分类名称、图标、颜色
- 支持分类排序
- 支持隐藏/删除自定义分类

**预设分类：**

| 支出分类 | 收入分类 |
|----------|----------|
| 餐饮美食 | 工资薪酬 |
| 交通出行 | 投资理财 |
| 购物消费 | 兼职副业 |
| 生活缴费 | 奖金红包 |
| 休闲娱乐 | 其他收入 |
| 医疗健康 | |
| 教育培训 | |
| 人情往来 | |
| 其他支出 | |

#### 4.2.5 账目列表

**功能描述：** 查看历史账目记录

**业务规则：**
- 按时间倒序排列
- 支持按月份筛选
- 支持按分类筛选
- 支持按成员筛选（家庭模式）
- 支持关键词搜索

**交互说明：**
- 列表展示：日期分组，每条显示分类图标、备注、金额
- 支持左滑删除/编辑
- 点击进入详情页

#### 4.2.6 统计报表

**功能描述：** 可视化展示收支数据

**业务规则：**
- 收支概览：本月收入、支出、结余
- 分类统计：各分类占比饼图
- 趋势图表：近6个月收支趋势折线图
- 成员贡献：各成员记账金额排行

**交互说明：**
- 支持切换时间范围（本周/本月/本年）
- 点击图表可查看详细数据
